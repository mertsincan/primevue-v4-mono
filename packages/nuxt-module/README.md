## Quick Setup

1. Add `nuxt-primevuelab` dependency to your project

```bash
npx nuxi@latest module add primevuelab
```

2. Add `nuxt-primevuelab` to the `modules` section of `nuxt.config.{ts,js}`

```js
{
  modules: [
    'nuxt-primevuelab'
  ]
}
```
