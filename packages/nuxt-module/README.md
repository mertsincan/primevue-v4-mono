## Quick Setup

1. Add `nuxt-primevue-lab` dependency to your project

```bash
npx nuxi@latest module add primevue-lab
```

2. Add `nuxt-primevue-lab` to the `modules` section of `nuxt.config.{ts,js}`

```js
{
  modules: [
    'nuxt-primevue-lab'
  ]
}
```
